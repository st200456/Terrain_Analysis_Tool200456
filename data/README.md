# Data inventory and provenance

## DEM Data 
- Source: OpenTopography – Continental Europe Digital Terrain Model (https://portal.opentopography.org/datasets)
- License: CC BY 4.0
- CRS: EPSG:4326 (WGS 84)
- Resolution: 30 m
- Files:
  output_be.tif, size ~90 KB, NoData = -9999.9
- Notes: DEM subset extracted via OpenTopography raster API for a user-defined bounding box covering a river reach.

## Landcover Data 
- Source: ESA WorldCover  (https://esa-worldcover.org/en/data-access)
- License: CC BY 4.0
- CRS: EPSG:4326 (WGS 84)
- Resolution: 10 m
- Files:
  output_be.tif, size ~90 KB  ,NoData = 0
- Notes: Tile for the ESA landcover is N48E009.
